RULES:
The objective is to move the entire stack of disks to any another rod in ascending order (the smallest at the top), obeying the following rules:
1. Only one disk from the top of the stack can be moved at a time.
2. No larger disk may be placed on top of a smaller disk.

CONTROLS:
Del  matches Reset button, returning game in initial state
Enter matches Solve!/Return button, turning auto solving on/off
←    matches < button, making step backward in the solution
→    matches > button, making step forward in the solution
<<  view all solution dynamically (backward)
>>  view all solution dynamically (forward)

Input the number of disks (4-10)
(current progress will be lost)
